My Resume

傻翠的简历生成引擎ww

🙋这是什么？

🤣写简历的时候才知道自己这么懒 QAQ
这是一个简历生成引擎，最初为Nodejs版本，后来包装成了网页，其主要特性有：
    • 根据传入的数据，自动生成美观简历
    • 避免了来回调整格式的困扰
    • 支持生成Markdown，HTML，PDF等多种形式的简历生成
    • 完全开源，可在此基础上自定义样式

🔧如何使用

🤣写简历的时候才知道自己这么菜 QAQ

在开始使用中，以上传文件或编辑的方式，将规定格式的数据放入其中，数据格式为：

            
{
    "name": "",
    "language": "zh-cn",
    "contact": {
        "address": "",
        "phone": "",
        "email": "",
        "blog": ""
    },
    "categories":{
        "类别1（如：教育背景）": [
            {
                "name": "项目1（如：南京大学商学院）",
                "time": "",
                "position": "",
                "details": [
                    "具体描述1",
                    "具体描述2",
                    ...
                ]
            },
            项目2，项目3...
        ],
        类别2，类别3...
    }
}

这个示例文件在这里，可以在此基础上进行改动，生成自己的简历文件
